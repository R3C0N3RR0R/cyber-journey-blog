# 🔑 Credential Cracking

## Qu'est-ce que le Credential Cracking ?

Le credential cracking consiste à deviner ou casser des identifiants (mots de passe, clés) pour accéder à un système ou un service.

## 🎯 Objectifs pédagogiques

- Comprendre les techniques de cassage de mots de passe
- Savoir utiliser des outils de brute force

## 📋 Exemples de scénarios et commandes

- Utiliser Hydra pour tester des mots de passe sur un service SSH
- Exploiter une fuite de hash pour casser un mot de passe avec John the Ripper
- Automatiser des attaques de brute force sur des formulaires web

## 🛡️ Bonnes pratiques

- Imposer des mots de passe forts et une politique de rotation
- Mettre en place une détection d'intrusion sur les tentatives de connexion
- Utiliser l'authentification multi-facteur

## 🛠️ Outils et commandes les plus utilisés

- **Hydra** : Brute force sur SSH, FTP, HTTP, etc.
  ```bash
  hydra -l admin -P wordlist.txt ssh://10.10.10.10
  hydra -L users.txt -P passwords.txt ftp://10.10.10.10
  ```
- **John the Ripper** : Casser des hashs
  ```bash
  john --wordlist=rockyou.txt hash.txt
  ```
- **Hashcat** : Casser des hashs GPU
  ```bash
  hashcat -m 0 -a 0 hash.txt rockyou.txt
  ```
- **Medusa** : Brute force multi-protocoles
  ```bash
  medusa -h 10.10.10.10 -u admin -P wordlist.txt -M ssh
  ```
- **CrackMapExec** : Attaques sur SMB/RDP
  ```bash
  crackmapexec smb 10.10.10.10 -u users.txt -p passwords.txt
  ```
